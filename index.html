Explain passed by value and passed by reference

    Passed by Value : It apply on the primtive data types. Copied variable assign different memory.
    ex: var a=5
    var b=a
    b=b+5
    console.log(b) Different memory and 5 Output
    console.log(a) Different memory and 10 Different Output

    Passed By Reference : It apply on the non-primitive data type. Copied datatype has same memory.
    ex: let arr1 = [1,2,3,4]
    let arr2=arr1 
    arr2.push(5)
    console.log(arr1)  //Output : 11,2,3,4,5] Same Memory Allocaton
    console.log(arr2)  //Output : 11,2,3,4,5] Same Memory Allocaton
